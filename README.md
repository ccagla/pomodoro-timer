
# Pomodoro Timer

A simple Pomodoro Timer built with HTML, CSS and Javascript.
Start , stop and reset a 25-minute focused work session using clean UI and intuitive UX.

## Features

- Start countdown (25 minutes)
- Pause / resume the timer
- Reset to the initial time
- Dynamic time update every second
- Clear and responsive user interface


## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Git & GitHub
